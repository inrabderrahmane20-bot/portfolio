# Portfolioo — Premium Creative Portfolio

A high-end Next.js portfolio experience built with:

- Next.js + TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- Lenis smooth scrolling

## Run locally

1. Install dependencies:

   `npm install`

2. Start development server:

   `npm run dev`

3. Open `http://localhost:3000`

## Pages included

- `/` — Homepage
- `/work` — Portfolio showcase
- `/work/[slug]` — Case study pages
- `/about` — About page
- `/contact` — Contact page
- `/admin` — Lightweight CMS preview dashboard
